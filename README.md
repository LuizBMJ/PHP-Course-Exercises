## 📌 Overview

This repository contains exercises and coding challenges developed during the PHP course from Curso em Vídeo, taught by Professor Gustavo Guanabara.
The project is organized into practical exercises and challenge-based activities designed to reinforce PHP fundamentals and backend development concepts.
It serves as a study repository focused on learning the core features of the PHP language through hands-on practice.

## 🛠️ Technologies

* **PHP** — Main language used for the exercises and challenges
* **Apache** — Web server used for running PHP applications
* **HTML5** — Structure and markup for web pages

## 🚀 Getting Started

Clone the repository:

```bash id="3dnf8s"
git clone https://github.com/LuizBMJ/php-course-exercises.git
```

Navigate to the project folder:

```bash id="x4n29k"
cd php-course-exercises
```

Run the PHP built-in server:

```bash id="7sl0qp"
php -S localhost:8000
```

Open your browser and access:

```text id="f4kd1o"
http://localhost:8000
```

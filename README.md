# PHP - Curso em Vídeo

Repositório com exercícios e desafios do curso de PHP do **Curso em Vídeo**, ministrado pelo professor Gustavo Guanabara.

## Sobre o Projeto

Este projeto contém materiais de estudo da linguagem PHP, organizados em duas pastas principais:

- **EXERCICIOS**: Exercícios práticos do curso
- **DESAFIOS**: Desafios propostos para fixação do conteúdo

- Configuração do ambiente PHP

## Como Executar

Para executar os arquivos PHP, você precisa de um servidor local (Apache, Nginx) ou usar o PHP built-in server:

```bash
php -S localhost:8000
```

Depois, acesso no navegador: `http://localhost:8000`

## Requisitos

- PHP 7.4 ou superior
- Servidor web (opcional, se usar PHP built-in)
- Navegador web
